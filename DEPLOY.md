# Moments - Deploy to Vercel/Netlify

## Quick Deploy

### Vercel
1. Connect your GitHub repository to Vercel
2. Set build command: `pnpm run build`
3. Set output directory: `apps/web-old/dist`
4. Deploy!

### Netlify
1. Connect your GitHub repository to Netlify
2. Set build command: `pnpm run build`
3. Set publish directory: `apps/web-old/dist`
4. Deploy!

## Local Development
```bash
pnpm install
pnpm run dev
```

## Production Build
```bash
pnpm run build
pnpm run preview
```
